# Axiom — an incremental game about numbers

A browser-based incremental game built around mathematics and hyperoperations.  
Start from the successor function. Build toward something that explains everything.

## Play

Open `index.html` in any modern browser. No installation required.

## Features

- Hyperoperation upgrade tree (Successor → Addition → Multiplication → Exponentiation → ...)
- Prestige layers: Axiom → Theorem → Conjecture → Paradox → Transcendence
- Sequences system with arithmetic and geometric progressions
- Minimalist aesthetic — monospace, sharp geometry, mathematical notation
